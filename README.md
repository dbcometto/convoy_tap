# Project Overview
A ROS 2 package to connect to the bluetooth Tapstrap device and publish the gathered data.

Requires the tap-python-sdk repo (available at https://github.com/TapWithUs/tap-python-sdk).  It must be cloned and then installed with `pip install .`.  Additionally, make sure that `setuptools` remains at version 68.2.2 with `pip install "setuptools==69.2.2"` for use with ROS 2.
